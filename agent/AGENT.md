## 🔄 交互流程

### 场景 1：记账

```
用户: "午饭 25"

Agent 思考:
1. 提取: 金额=25, 分类=餐饮, 备注=午饭
2. 分类推断: "午饭" → "餐饮"
3. 构建: cang fin tx add --amount 2500 --account 1 --category 餐饮 --note "午饭"
4. 执行: (调用 CLI)
5. 响应: "囤粮成功！🐹 已记录：午饭 25 元"
```

### 场景 2：查询净资产

```
用户: "我有钱吗"

Agent 思考:
1. 意图: 查询资产总额
2. 构建: cang asset networth
3. 执行: (调用 CLI)
4. 响应: "主人现在值 💰 1,380,000 元"
```

### 场景 3：大额消费提醒

```
用户: "买了台电脑 8000"

Agent 思考:
1. 检测: 金额 > 500，大额消费
2. 确认: "嘶...这块骨头有点大 🦴"

```

### 1. 命令构建

```python
def build_tx_add_command(amount: float, category: str, note: str) -> str:
    account_id = get_default_account()  # 需要实现
    amount_cents = int(amount * 100)
    return f'cang fin tx add --amount {amount_cents} --account {account_id} --category {category} --note "{note}"'
```

### 2. CLI 调用

```python
import subprocess
import json

def call_cli(command: str) -> dict:
    result = subprocess.run(
        command.split(),
        capture_output=True,
        text=True
    )
    return json.loads(result.stdout)
```

### 3. 智能分类

```python
AUTO_CATEGORIES = {
    "饭": "餐饮", "餐": "餐饮", "吃": "餐饮",
    "地铁": "交通", "打车": "交通", "加油": "交通",
    # ...
}

def infer_category(note: str) -> str:
    for keyword, category in AUTO_CATEGORIES.items():
        if keyword in note:
            return category
    return "其他"
```
