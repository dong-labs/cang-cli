# 仓咚咚 (Cang) - Agent 架构

> 基于 Cang CLI 的 AI 原生个人财务智能体

---

## 🐹 Agent 简介

仓咚咚是一只爱囤钱的小仓鼠，负责帮主人管理个人财务。

**核心理念**：数据完全通过 CLI 操作，Agent 负责理解和交互。

## 🏗️ 三层架构

```
┌─────────────────────────────────────────────────────────┐
│                       用户交互层                           │
│                   自然语言 / 消息输入                        │
└────────────────────┬────────────────────────────────────┘
                     │
┌────────────────────┴────────────────────────────────────┐
│                      Agent 智能体层                         │
│  ┌──────────────────────────────────────────────────┐    │
│  │  理解意图 → 自动分类 → 构建命令 → 调用 CLI     │    │
│  │  解析结果 → 添加口癖 → 格式化输出 → 返回用户   │    │
│  └──────────────────────────────────────────────────┘    │
└────────────────────┬────────────────────────────────────┘
                     │
┌────────────────────┴────────────────────────────────────┐
│                      CLI 工具层                            │
│  cang fin tx add │ cang asset networth │ cang invest profit │
│  所有命令返回 JSON，结构化数据，易于 AI 解析            │
└────────────────────┬────────────────────────────────────┘
                     │
┌────────────────────┴────────────────────────────────────┐
│                      数据存储层                            │
│              ~/.cang/cang.db (SQLite)                     │
└─────────────────────────────────────────────────────────┘
```

## 🚀 快速开始

### 1. 安装 CLI

```bash
pip install -e .
```

### 2. 初始化数据库

```bash
cang fin init
```

### 3. 测试命令

```bash
# 记账
cang fin tx add --amount 2500 --account 1 --category 餐饮 --note "午饭"

# 查余额
cang fin account balance

# 查净资产
cang asset networth
```

### 4. 集成到 AI

将 `PROMPT.md` 的内容作为 System Prompt，即可获得仓咚咚的完整能力。
