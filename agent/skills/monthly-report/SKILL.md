# 月度消费报告

## 功能

生成本月消费汇总分析。

## CLI 命令

```bash
cang fin tx summary --period month
cang fin tx ls --start <月初> --end <月底>
```

## 输出格式

```
仓咚咚：3月消费报告 📊

| 分类 | 金额 | 占比 |
|------|------|------|
| 房贷 | 8,000 | 56% |
| 餐饮 | 1,823 | 13% |
| 购物 | 3,291 | 23% |
| 交通 | 456 | 3% |
| 娱乐 | 680 | 5% |
| **总计** | **14,250** | 100% |

对比：
- 比上月多 1,200（购物多了）
- 主要大额：羽绒服 900

建议：
- 购物预算设 2,000？
```
