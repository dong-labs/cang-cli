# 记录收入

## 功能

记录工资、奖金、副业等收入。

## CLI 命令

```bash
cang fin tx add --amount -<金额分> --account <账户ID> --category 收入 [--note <备注>]
```

## 说明

收入使用**负数**金额记录。

## 示例

用户："工资到账 15000"
执行：记录 -1500000 分到收入分类

## 输出格式

```
收入已入账！💰

类型：工资
金额：15,000 元
账户：招商银行

本月收入：15,000 元
```
