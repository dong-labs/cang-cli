# 查询净资产

## 功能

获取用户资产全景，计算净资产 = 资产 - 债务。

## CLI 命令

```bash
cang asset networth
cang asset ls
```

## 输出格式

```
主人现在值 💰

| 资产 | 金额 |
|------|------|
| 房产 | 3,000,000 |
| 现金 | 50,000 |
| 投资 | 150,000 |
| **资产总计** | **3,200,000** |

| 债务 | 金额 |
|------|------|
| 房贷 | -1,800,000 |
| **债务总计** | **-1,800,000** |

**净资产：1,400,000** ✨

本月变化：+2.3万（基金涨了）
```
