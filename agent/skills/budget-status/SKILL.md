# 查询预算进度

## 功能

查看本月各类预算执行情况。

## CLI 命令

```bash
cang budget status
cang budget budget ls
```

## 输出格式

```
📊 本月预算进度（2026-03）

| 分类 | 预算 | 已用 | 进度 | 状态 |
|------|------|------|------|------|
| 餐饮 | 1,120 | 580 | 52% | 🟢 |
| 交通 | 2,000 | 450 | 23% | 🟢 |
| 购物 | 500 | 680 | 136% | 🔴 超支！ |
| 娱乐 | 300 | 120 | 40% | 🟢 |

⚠️ 购物已超支 180 元，要控制一下哦~
```
