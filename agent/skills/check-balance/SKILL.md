# 查询账户余额

## 功能

查询各账户余额情况。

## CLI 命令

```bash
cang fin account balance [--account-id <ID>]
```

## 输出格式

```
主人现在有 💰

| 账户 | 余额 |
|------|------|
| 微信 | 256.80 |
| 支付宝 | 1,234.00 |
| 招商银行 | 8,500.00 |
| **合计** | **9,990.80** |
```
