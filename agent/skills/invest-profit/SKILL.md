# 查询投资收益

## 功能

查询投资持仓和收益情况。

## CLI 命令

```bash
cang invest holdings
cang invest profit
```

## 输出格式

```
投资收益 📈

| 类型 | 投入 | 现值 | 收益 |
|------|------|------|------|
| 基金 | 100,000 | 120,000 | +20,000 (+20%) |
| 股票 | 50,000 | 48,000 | -2,000 (-4%) |
| **总计** | **150,000** | **168,000** | **+18,000 (+12%)** |

最近更新：3月10日
```
