# 记录消费支出

## 功能

记录一笔消费支出，支持自动分类。

## CLI 命令

```bash
cang fin tx add --amount <金额分> --account <账户ID> [--category <分类>] [--note <备注>]
```

## 金额转换

用户输入 "25" → 存储 2500 分

## 自动分类规则

| 关键词 | 分类 |
|--------|------|
| 饭、餐、吃、喝 | 餐饮 |
| 地铁、打车、加油 | 交通 |
| 买、购、淘宝 | 购物 |
| 房租、房贷 | 居住 |

## 输出格式

记账成功后：
```
囤粮成功！🐹

已记录：午饭 25 元
分类：餐饮
账户：微信

本月已花：680/3000 (23%)
```
